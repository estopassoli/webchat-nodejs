# Webchat em NodeJs

Este é um simples projeto de um webchat com WebSocket.

# Utilizado

`yarn v1.22.17`
`node v12.21.0`
`npm v6.14.11`

# Getting Start

Para iniciar um projeto em NodeJs
`npm init -y`

Dependências:
(utilizei yarn para instalar, pode ser instalado também com npm install)
`yarn add ejs`
`yarn add express`
`yarn add socket.io`
